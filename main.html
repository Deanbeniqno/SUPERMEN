Sistem Informasi Utama Personel Metrologi Legal Nasional

Deskripsi Sistem

Tujuan Utama

Sistem terintegrasi untuk pengendalian personel metrologi legal, memfasilitasi pendaftaran, verifikasi, dan monitoring.

Target Pengguna
  • Pemerintah Daerah & Dinas
  • Petugas Metrologi
  • Verifikator & Direktorat
  • Administrator Pusat
Fitur Utama
  • Pendaftaran Online
  • Dashboard Real-time
  • Manajemen Sertifikasi
  • Pelaporan Terintegrasi
  • Verifikasi Email

Technology Stack

Frontend HTML5 CSS3 JavaScript Bootstrap
Backend Python Flask JWT SendGrid
Database MySQL Redis
Infrastructure AWS Apache SSL

Struktur Database Sistem SIPPL

Tabel Personel

FieldType
idUUID PRIMARY KEY
nipVARCHAR(18) UNIQUE
namaVARCHAR(255)
jabatanVARCHAR(100)
emailVARCHAR(255) UNIQUE
status_sertifikasiENUM(aktif, kadaluarsa)
created_atTIMESTAMP

Tabel Sertifikasi

FieldType
idUUID PRIMARY KEY
personel_idUUID FOREIGN KEY
jenis_sertifikasiVARCHAR(100)
tanggal_terbitDATE
tanggal_kadaluarsaDATE
status_verifikasiENUM(diverifikasi, belum)
created_atTIMESTAMP

Tabel Audit Logs

FieldType
idUUID PRIMARY KEY
personel_idUUID FOREIGN KEY
aksiVARCHAR(100)
waktuTIMESTAMP
old_valuesJSONB
new_valuesJSONB

Relasi Database

  • Personel → Sertifikasi: One-to-Many
  • Personel → Audit Logs: One-to-Many
